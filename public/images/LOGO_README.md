# Logo Setup Instructions

## Required Logo File

Please save your Voss Taxi logo as:

**File path:** `/public/images/voss-taxi-logo.png`

## Specifications

- **Format:** PNG
- **Recommended size:** 400-600px width
- **Background:** Transparent recommended
- **Color mode:** RGB

## Current Usage

The logo is used in:
1. PDF receipts (`/components/receipts/ReceiptPDF.tsx`)
2. Website pages (as needed)

## How to Add

1. Save your logo image as `voss-taxi-logo.png`
2. Place it in the `/public/images/` directory
3. The logo will automatically appear in PDF receipts

## Note

The code is already configured to use this logo. Just save your logo file with the correct name and it will work immediately.
